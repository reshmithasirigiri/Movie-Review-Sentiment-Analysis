# 🎬 Movie Review Sentiment Analysis

## Project Overview

This project predicts whether a movie review is **Positive** or **Negative** using Natural Language Processing (NLP) and Machine Learning.

## Dataset

- IMDb Movie Review Dataset
- 50,000 Movie Reviews
- Positive and Negative Sentiment

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLTK
- Scikit-learn

## Machine Learning Algorithm

- Logistic Regression

## Workflow

1. Load Dataset
2. Data Cleaning
3. Text Preprocessing
4. TF-IDF Vectorization
5. Train-Test Split
6. Logistic Regression Model
7. Model Evaluation
8. Custom Prediction

## Results

- Accuracy: Around 90%
- Confusion Matrix
- Classification Report
